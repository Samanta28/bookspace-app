# BookSpace

Projekt systemu do zarządzania książkami, listami czytelniczymi i recenzjami.

## Funkcjonalności (MVP)
- logowanie i rejestracja
- dodawanie książek
- lista książek
- recenzje
- lista "do przeczytania"

## Architektura
- Auth Service
- Book Service
- PostgreSQL

## Technologie
- FastAPI
- Flutter Web
- PostgreSQL

## Status
Projekt w fazie projektowania.
