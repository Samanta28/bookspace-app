# BookSpace

Projekt systemu do zarządzania książkami, listami czytelniczymi i recenzjami.

## Funkcjonalności (MVP)
- logowanie i rejestracja
- dodawanie książek
- lista książek
- recenzje
- lista "do przeczytania"

## Architektura
- Auth Service (FastAPI)
- Book Service (FastAPI)
- PostgreSQL

## Technologie
- FastAPI
- Flutter Web
- PostgreSQL

## Status
Projekt w fazie projektowania.

## Git workflow
Projekt korzysta z branchy: main → develop → feature/*
